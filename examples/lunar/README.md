# examples/lunar

Lunar DEM and CERES / small-body kernelcal drivers.

| Script                | What it does                                                                          |
| --------------------- | ------------------------------------------------------------------------------------- |
| `lunar_kernelcal.py`  | Spectral-kernel analysis on lunar DEM patches (Robbins crater catalog overlay).       |
| `lunar_viz.py`        | Visualization helpers for lunar kernel outputs.                                        |

Figures land in `lunar_figures/` and `lunar_paper_figures/` at the repo root.
