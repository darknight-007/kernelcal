# examples/urban

OpenStreetMap street-network spectral analyses.

| Script                     | What it does                                                                     |
| -------------------------- | -------------------------------------------------------------------------------- |
| `osm_street_kernelcal.py`  | Single-city street-graph spectral analysis (nodes = intersections).              |
| `osm_urban_kernelcal.py`   | Multi-city urban-form comparison; bootstrap null tests; controller ranking.      |

Cache lives in `urban_cache/` at the repo root (OSMnx extractions).
Figures land in `urban_figures/` and `street_figures/` at the repo root.
