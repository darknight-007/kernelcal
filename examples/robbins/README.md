# examples/robbins

Robbins lunar-crater catalog analysis and paper-figure drivers.

| Script                              | What it does                                                                           |
| ----------------------------------- | -------------------------------------------------------------------------------------- |
| `robbins_kernelcal.py`              | Spectral-kernel analysis over the Robbins crater catalog.                              |
| `robbins_paper_figs.py`             | Publication-grade figures for the Robbins section of the paper.                        |
| `make_robbins_phasespace_figure.py` | Phase-space diagram for Robbins catalog controller sweep.                              |

Figures land in `lunar_paper_figures/` at the repo root.
