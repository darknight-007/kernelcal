# examples/controller

MaxCal-driven kernel-controller experiments.

| Script                                    | What it does                                                                           |
| ----------------------------------------- | -------------------------------------------------------------------------------------- |
| `drone_dem_betti_adaptive_experiment.py`  | Drone DEM adaptive exploration with β₀/β₁/Fiedler-driven quadrant selection.           |
| `make_phasespace_figure.py`               | Rendering of the kernelcal phase-space figure used in the paper.                        |
| `route3_conservation_test.py`             | Route-3 conservation-law test across P_8 parameters.                                    |
| `run_a2_sweep.py`                         | A² coupling sweep across a grid of Lagrange multipliers.                                |

Figures land in `drone_dem_figures/` and `terrain_figures/` at the repo root.
